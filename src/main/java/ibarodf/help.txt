HELP

    DESCRIPTION
        This program allows you to view the metadata of ODT files and edit some of them.
    

    CONTROLS
    -h ou --help
        display help

    -f [file_name] or --file [file_name]
        If the file is an ODT file : displays its metadata.
        Else display the name and the type of the file.

        Command options ONLY FOR ODT FILES 
            -t [new_file_name] or --title [new_file_name]
                Change the file's name.
            -s [new_subject] or --subject [subject]
                Change the subject.
            -k [new_keyword] or --keyword [keyword]
                Add a new keyword.
            -co [new_comments] or --comments [new_comments]
                change the comments.
            -cr [creator_name] or --creator [creator_name]
                Change the creator.
            -de or --description 
                Replaces the previous description with the new one.
    
    -d [path] or --directory [path]
        Displays metadata for ODT files in a directory.
